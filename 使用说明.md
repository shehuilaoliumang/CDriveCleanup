# 使用说明

## 启动程序

双击 `启动.bat` 启动。脚本会优先使用项目内虚拟环境。

如果双击无反应，可以在项目目录打开 PowerShell 后运行：

```powershell
.\.venv\Scripts\python.exe main.py
```

## 主界面说明

界面顶部显示标题、主题按钮和系统性能信息。中间是功能标签页。右侧滚动条用于在低分辨率或窗口较小时查看完整内容，内容宽度会跟随窗口自动适配。

如果看到空白或未响应，请确认当前版本已经包含滚动容器修复：`main.py` 中不应再出现 `main_frame.config(width=...)`。

## 磁盘扫描

1. 进入“磁盘扫描”标签页。
2. 选择扫描路径，默认是 `C:\`。
3. 点击“开始扫描”。
4. 扫描结束后查看大文件列表、文件类型统计和历史记录。

支持停止扫描、恢复扫描和增量扫描。扫描大目录时耗时较长是正常现象。

## 安全清理

1. 进入“安全清理”标签页。
2. 勾选需要清理的项目。
3. 可先点击“预览文件列表”或“计算可清理空间”。
4. 确认后点击“开始清理”。

“计算可清理空间”现在会在后台执行，界面不会因为计算大量缓存文件而卡住。

## 智能清理建议

智能清理建议不会在启动时自动计算。需要时点击“刷新建议”，程序会在后台统计可释放空间。

“一键清理安全项”会使用最近一次刷新得到的建议。如果尚未刷新，会先提示刷新建议。

## 重复文件

1. 进入“重复文件”标签页。
2. 选择扫描目录。
3. 点击“查找重复文件”。
4. 根据结果选择文件，可以移动或删除。

重复文件检测会计算文件哈希，处理大目录时会比较耗时。

## 磁盘可视化

该功能需要 `matplotlib`。选择目录并点击“分析磁盘”后，可以查看饼图、树状图、扩展名分布和文件分类统计。

## 白名单和备份

- 白名单中的文件不会被清理。
- 启用“清理前备份文件”后，清理前会复制文件到备份目录。
- 可以在“文件恢复”标签页查看和恢复备份文件。

## 定时清理

定时清理会在后台运行，不会阻塞主界面。建议只选择安全项，例如临时文件和浏览器缓存。

定时清理支持“每 N 天”执行一次：

- 每天：执行间隔填 `1`。
- 每周：执行间隔填 `7`，或点击“每周”。
- 每 10 天：执行间隔填 `10`。
- 半个月：执行间隔填 `15`，或点击“半月”。

程序会记录上次触发日期，避免同一天重复执行。定时计划会保存到 `app_settings.json`。

## 常见问题

### 界面空白或未响应

通常是旧版滚动布局造成的。当前版本已修复 Canvas/Frame 的循环布局问题。

### 清理后空间变化不明显

部分文件可能进入回收站，或者被系统占用。可以清空回收站，或以管理员身份重新运行。

### 测试产生的 tmp 目录删不掉

这是测试失败时遗留的权限异常目录。需要管理员权限删除，或重新获取目录所有权后删除。

### 依赖缺失

核心扫描和清理功能使用标准库。可视化、托盘、打包和性能监控需要可选依赖，详见 `requirements.txt`。
